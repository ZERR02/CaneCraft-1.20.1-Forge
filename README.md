**CaneCraft** is a mod that expands sugarcane into a full crafting system — from tools and armor to food and magical experience.

> ⚠️ **Note:** This is my first mod. The code may be a bit messy in places, but everything works. Feedback and suggestions are welcome!

## 📖 Features

- **New Items**
  - Bagasse (fuel source and craft item)
  - Stem sticks, cobble sticks
  - Mortar & Pestle (durable crafting tool)
  - Cane juice, cane syrup, caramel, caramel apples and berries

- **Tools & Weapons**
  - Cane Sword
  - Cane Pickaxe, Axe, Shovel, Hoe
  - Full Cane Armor

- **Special Effect**
  - Sugar Overload — negative effects of eating sugary foods
